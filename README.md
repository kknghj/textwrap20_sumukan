# 필사 연습장 줄바꿈 변환기

시, 편지, 산문 등의 원문을 필사 연습장에 맞게 줄바꿈해 주는 브라우저 전용 웹앱입니다.
입력한 텍스트는 서버로 전송되지 않으며, 모든 변환은 사용자의 브라우저 안에서만 처리됩니다.

## 앱 방향

**최종 목표는 HWPX 전용**입니다. Excel을 거치지 않고 브라우저에서 `.hwpx` 연습장 파일을 바로 다운로드해 한글에서 열어 필사하는 흐름이 핵심입니다.

```text
원문 입력
→ 줄바꿈 변환
→ 한글 연습장(.hwpx) 다운로드   ← 최종 목표 (준비 중)
→ 한글에서 바로 열어 필사
```

Excel 다운로드는 HWPX 기능이 안정화되기 전까지만 **레거시/임시**로 남아 있으며, **최종적으로 제거**할 예정입니다.

## 기능 상태

| 구분 | 내용 |
| --- | --- |
| **현재 지원** | 텍스트 줄바꿈 변환, 변환 결과 복사, 한글 연습장 붙여넣기용 표 미리보기 |
| **준비 중** | 대각선 20×29 HWPX 연습장 다운로드 (브라우저에서 `.hwpx` 직접 생성, **한 페이지 580칸**) |
| **임시 지원 (제거 예정)** | Excel 다운로드 → 엑셀 복사 → 한글 붙여넣기 (HWPX 안정화 전까지만 유지) |

HWPX 다운로드 기능은 아직 구현되지 않았습니다. 설계 문서는 [`docs/hwpx_download_plan.md`](docs/hwpx_download_plan.md)를 참고하세요.

## 주요 기능 (현재)

- 한 줄당 최대 글자 수 설정 (최소 10자) — HWPX 모드에서는 20칸으로 고정 예정
- 글자 수 계산 방식: 띄어쓰기 제외 / 포함
- 줄넘김 기준: 어절 단위 / 글자 단위
- 출력 형식: 기본 텍스트 / 한글 연습장 붙여넣기 (표 미리보기) → **향후 HWPX 전용 모드로 전환**
- 문장 부호, 괄호, 따옴표, 한자, 영어 등 제거 옵션
- 원문과 변환 결과를 나란히 확인 (변환 결과 줄 수 표시)
- 변환 결과 복사 및 초기화

## 한글 연습장 연동

### 목표: HWPX 다운로드 (준비 중)

Excel을 거치지 않고, 브라우저에서 대각선 **20×29** 연습장 양식이 채워진 `.hwpx` 파일을 바로 다운로드하는 기능을 개발 중입니다.
템플릿은 `assets/templates/practice_diagonal_20x29/`에 준비되어 있습니다.

#### MVP 제한 (설계 확정)

- **한 페이지 580칸**(20열 × 29행)까지만 지원
- HWPX 다운로드 모드에서는 **한 줄 글자 수를 20자로 고정** (다른 설정값이 있어도 20×29 템플릿 기준으로 생성)
- 580칸 초과 시 다운로드 차단 및 안내 문구 표시

권장 안내 문구:

```text
현재 HWPX 다운로드 MVP는 한 페이지(580칸)까지만 지원합니다.
변환 결과가 580칸을 초과했습니다.
다중 페이지 지원은 다음 버전에서 추가할 예정입니다.
```

#### UI 방향 (구현 예정)

HWPX 모드 선택 시 `maxCharsPerLine` 입력을 20으로 고정하거나, HWPX 다운로드 버튼 옆에 **「20칸 템플릿 기준」** 안내를 표시합니다.

후속 버전에서는 16칸·19칸·20칸 등 **다중 템플릿** 지원을 검토합니다. (MVP 범위 밖)

### 임시 방식: Excel 다운로드 (레거시)

HWPX 다운로드가 준비·안정화되기 전까지, **출력 형식 → 한글 연습장 붙여넣기** 모드에서 Excel 다운로드를 임시로 사용할 수 있습니다.
이 방식은 **최종 제거 예정**이며, HWPX 전용 흐름으로 대체됩니다.

1. **출력 형식 → 한글 연습장 붙여넣기** 선택
2. 원문 입력 후 **변환**
3. **Excel 다운로드** 클릭 → `pilsa-practice.xlsx` 저장
4. Excel에서 파일을 열고 표 전체 선택 후 **복사(Ctrl+C)**
5. 한글 글씨 연습장 표의 **첫 칸에 커서를 두세요** (표 전체를 드래그해 선택하지 마세요)
6. **붙여넣기(Ctrl+V)** → **내용만 덮어쓰기** 선택  
   - 안 되면 **편집 → 골라 붙이기 → Microsoft Excel 워크시트** 선택

| 항목 | 내용 |

## 로컬 실행 방법

```bash
pnpm install
pnpm dev
```

브라우저에서 Vite가 안내하는 로컬 주소(예: `http://localhost:5173`)로 접속합니다.

## 테스트 실행 방법

```bash
pnpm test
```

watch 모드로 실행하려면:

```bash
pnpm test:watch
```

## 빌드 방법

```bash
pnpm build
```

빌드 결과는 `dist` 폴더에 생성됩니다. 로컬에서 빌드 결과를 확인하려면:

```bash
pnpm preview
```

## 향후 배포 방법

Vercel 정적 사이트 배포를 기준으로 아래 순서를 사용할 수 있습니다.

1. GitHub 저장소에 프로젝트를 푸시합니다.
2. [Vercel](https://vercel.com)에서 새 프로젝트를 생성하고 저장소를 연결합니다.
3. Framework Preset은 `Vite`를 선택합니다.
4. Build Command: `pnpm build`
5. Output Directory: `dist`
6. Install Command: `pnpm install`

배포 후에도 앱은 정적 파일만 제공하므로 별도 서버 없이 동작합니다.

## 개인정보 안내

- 입력한 원문과 변환 결과는 브라우저 메모리에서만 사용됩니다.
- 외부 API 호출이나 서버 전송 로직은 포함되어 있지 않습니다.
- 페이지를 새로고침하거나 초기화하면 입력 내용이 사라집니다.
