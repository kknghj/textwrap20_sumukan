# 한글 연습장 생성기 SUMUKAN

스무은 글씨를 더 반듯하게 쓰고 싶은 사람을 위한 한글 연습장 자동 생성기입니다.
연습하고 싶은 문장을 붙여넣으면 20×29칸 한글 연습장(.hwpx)으로 변환하고, 대각선·방안선·십자선 가이드 중 원하는 양식을 선택해 바로 다운로드할 수 있습니다.
입력한 텍스트는 서버로 전송되지 않으며, 모든 처리는 사용자의 브라우저 안에서만 이루어집니다.

## 주요 기능

- **`.hwpx 연습장 다운로드`** — 대각선·방안선·십자선 20×29 템플릿에 변환 결과를 채워 한글 파일 생성
- 580칸마다 자동 페이지 분할 (최대 20페이지 / 11,600칸)
- 줄바꿈 미리보기 및 결과 복사
- 글자 수 계산 방식: 띄어쓰기 제외 / 포함
- 줄넘김 기준: 어절 단위 / 글자 단위
- 문장 부호, 괄호, 따옴표, 한자, 영어 등 제거 옵션

## 사용 방법

1. 원문을 입력하고 **변환** 클릭
2. **연습장 종류** 선택 (대각선 / 방안선 / 십자선)
3. **`.hwpx 연습장 다운로드`** 클릭
4. 한글에서 파일을 열어 필사

HWPX 다운로드는 **20칸 × 29줄** 기준으로 생성됩니다. 미리보기의 줄 글자 수 설정과는 별도입니다.

| 템플릿 | 설명 |
| --- | --- |
| 대각선 | 대각선 가이드 필사 연습장 (기본) |
| 방안선 | 정사각 격자/방안선 필사 연습장 |
| 십자선 | 칸 중앙 십자선 가이드 필사 연습장 |

## 템플릿 출처

연습장 HWPX 템플릿은 **디시인사이드 문방구 마이너 갤러리**의 **BANOLIM**님이 무료 배포한 필사 연습장 양식을 기반으로 합니다.

- 상업적 이용: **하지 않음**
- 배포 범위: 무료 배포 가능 범위 내에서 사용

## 로컬 실행

```bash
pnpm install
pnpm dev
```

## 테스트

```bash
pnpm test
```

## 빌드

```bash
pnpm build
```

## Vercel 배포

1. GitHub 저장소에 프로젝트를 푸시합니다.
2. [Vercel](https://vercel.com)에서 새 프로젝트를 생성하고 저장소를 연결합니다.
3. Framework Preset: **Vite**
4. Build Command: `pnpm build`
5. Output Directory: `dist`
6. Install Command: `pnpm install`

## 개인정보

- 입력한 원문과 변환 결과는 브라우저 메모리에서만 사용됩니다.
- 외부 API 호출이나 서버 전송 로직은 포함되어 있지 않습니다.

## 기술 스택

- React, Vite, TypeScript
- Vitest
- fflate (HWPX ZIP 생성)
